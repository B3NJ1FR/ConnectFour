                                                                                                             
		    > ------------------- < CONNECT FOUR > ------------------- <


- Date : 28/02/2021
- Author : ARTHAUD Benjamin


- Description :
	
	Connect four, ou Puissance 4, est un jeu de stratégie dont l'objectif
est d'aligner une suite 4 jetons de votre couleur, de manière horizontale, verticale ou en diagonale,
sur une grille de 6 rangées et 7 colonnes.
	Pour jouer, il suffit de faire clique gauche à tour de rôle, 
sur la colonne voulue pour ajouter un jeton, afin qu'il tombe jusqu'à la position la plus basse possible.

! NB ! : - Il faut attendre 2 secondes à chaque fois afin de jouer après l'adversaire,
	   qu'il soit un autre joueur ou une IA.
 	 - L'IA difficile est la plus intéressante contre laquelle jouer puisqu'elle possède
           les fonctions d'évalutations les plus poussées du jeu


- Contrôles par défaut : 

	- Jouer un jeton = 			Clique Gauche
	- Quitter le jeu = 			Bouton Quitter du Menu Principal / Alt + F4
